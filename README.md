## どうやる？
  
![texture1](https://github.com/eleonora2201/texture_paint//Images/image3.png)  
右の3Dの状態から左の2Dのように表面を広げる  
(これは僕がやります)  

![texture2](https://github.com/eleonora2201/texture_paint/Images/image4.png)
